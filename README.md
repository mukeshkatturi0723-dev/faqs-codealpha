# 🤖 AI FAQ Chatbot

An intelligent FAQ Chatbot developed as part of the CodeAlpha Internship Program. This web application helps users quickly find answers to frequently asked questions through an interactive chatbot interface, improving user experience and reducing manual support efforts.

🔗 https://faqs-codealpha.surge.sh/

## ✨ Features

- 🤖 AI-powered FAQ chatbot
- 💬 Interactive chat interface
- ⚡ Instant responses
- 🔍 Smart FAQ search functionality
- 📱 Responsive design for all devices
- 🎨 Clean and modern UI
- 🚀 Fast and lightweight performance
- 🛠 Easy FAQ management
